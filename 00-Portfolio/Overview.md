# 作品概览

## 1) 监控告警（Zabbix）
- 覆盖：CPU/内存/磁盘/网络/端口/关键进程或服务
- 产出：告警截图 + 事件闭环记录（见 Incident-Examples）

## 2) 自动化运维（Ansible）
- 覆盖：基础工具安装、Docker 部署、配置分发、服务启停
- 产出：inventory + playbooks + 执行结果截图

## 3) 容器化部署（Docker/Compose + Nginx）
- 覆盖：compose 编排、反向代理、健康检查、日志查看
- 产出：一键启动 demo + 排障说明

## 4) 虚拟化（VMware）
- 覆盖：VM创建/快照/克隆；NAT/桥接网络设置与使用场景
- 产出：配置截图 + 操作记录
