# ops-playbook - IT运维与DevOps实践记录

本仓库用于展示我的 IT 运维与 DevOps 学习与实践成果（面向传统企业IT/运维开发实习）：
- 监控告警：Zabbix（主机监控、告警策略、事件闭环）
- 自动化运维：Ansible（批量执行、配置分发）、Shell/Python 巡检脚本
- 容器化部署：Docker / docker-compose、Nginx 反向代理
- 虚拟化：VMware（VM创建、快照、NAT/桥接网络）
- 故障处理：常见 Runbook（CPU高、磁盘满、服务不可用、DNS异常等）
